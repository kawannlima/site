O que é um software?

conjunto de componentes lógicos de um computador ou sistema de processamento de dados; programa, rotina ou conjunto de instruções que controlam o funcionamento de um computador; suporte lógico.

O que é um sistema operacional?

é um software fundamental que atua como intermediário entre o hardware e os programas (aplicativos) em um computador.

O que é uma licença de software?

é um contrato legal que define os termos e condições de uso, cópia, modificação e distribuição de um software.

O que é um Software Licenciado?

é um documento contratual usado por desenvolvedores de programas digitais.

O que é um Software livre?

programas de computador que permitem aos utilizadores liberdade para executar, copiar, distribuir, estudar, modificar e melhorar o software.

O que é um Software Proprietário?

é aquele que pertence a uma pessoa ou empresa e está sujeito a contratos de licença que limitam seu uso, modificação e distribuição.

O que é um software de Sistema?

conjunto de programas que fornece a base para a operação de outros softwares, como aplicativos e utilitários.

O que é um software Aplicativo?

é um programa de computador projetado para executar tarefas específicas e úteis para o usuário final.

O que é um software de Programação?

conjunto de ferramentas que os programadores utilizam para criar, testar e debugar outros softwares.

O que é um software de diagnóstico de software ?

conjunto de ferramentas que os programadores utilizam para criar, testar e debugar outros softwares.

O que é um software de diagnóstico de hardware ?

conjunto de ferramentas que os programadores utilizam para criar, testar e debugar outros softwares.
