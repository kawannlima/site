Windows 1.0 (1985) – Primeira versão com interface gráfica baseada em janelas.

O Windows 1.0, lançado em 1985, foi a primeira versão do sistema operacional Windows da Microsoft a apresentar uma interface gráfica baseada em janelas.

Windows 3.1 (1992) – Muito popular, trouxe suporte a fontes TrueType e multitarefa.

O Windows 3.1, lançado em 1992, foi uma versão muito popular do sistema operacional, conhecida por introduzir fontes TrueType e melhorar a capacidade de multitarefa, tornando o Windows mais atraente para um público mais amplo, especialmente no setor de publicação.

Windows 95 (1995) – Introduziu o menu Iniciar e a barra de tarefas.

O Windows 95, lançado em 1995, foi a primeira versão do Windows a introduzir o menu "Iniciar" e a barra de tarefas, que se tornaram elementos centrais da interface do Windows e de outros sistemas operacionais. Esses recursos facilitaram a navegação e o acesso aos programas, tornando a experiência do usuário mais intuitiva.

Windows ME (Millennium Edition) (2000) – Focado em multimídia e uso doméstico, mas criticado por sua instabilidade.

O Windows Millennium Edition (ou Windows ME) é uma versão do Windows reconhecida como um dos maiores fracassos da Microsoft. O sistema foi lançado há 20 anos, em 14 setembro de 2000, e é famoso pelas telas azuis e por travar com uma simples mexida no mouse.

Windows NT 4.0 (1996) – Voltado para empresas, com interface parecida com o 95.

O Windows NT 4.0, lançado em 1996, era um sistema operacional voltado para empresas e servidores, com uma interface gráfica que se assemelha à do Windows 95.

Windows 2000 (2000) – Muito usado em servidores, foi base para o XP.

O Windows 2000 (lançado em 2000) foi, de facto, um sistema operacional muito usado em servidores, e também foi a base para o desenvolvimento do Windows XP.

Windows XP (2001) – Um dos sistemas mais populares da história. Estável, visual amigável.

O Windows XP foi uma versão do Microsoft Windows, uma série de sistemas operacionais desenvolvidos pela Microsoft para computadores pessoais, incluindo computadores residenciais e de escritório, notebooks, tablets e media centers.

Windows Vista (2007) – Visual moderno, mas teve críticas por desempenho.

O Windows Vista, lançado em 2007, apresentava um visual moderno, mas foi criticado por seu desempenho, que era considerado lento em comparação com o Windows XP, mesmo em computadores com configuração de hardware similar.

Windows 7 (2009) – Sucesso após o Vista. Rápido, leve e confiável.

A Microsoft está com sua mais nova tacada nas mangas. É o Windows 7, que já pode ser considerado o sistema operacional da turma de Bill Gates como o mais antecipado e esperado de toda a história dessa gigante da informática.

Windows 8 (2012) – Foco em telas sensíveis ao toque. Mudança polêmica no Menu Iniciar.

O Windows 8, lançado em 2012, introduziu uma interface "Metro" focada em telas sensíveis ao toque, substituindo o tradicional Menu Iniciar. Essa mudança foi amplamente criticada pelos usuários, gerando um mercado de aplicativos e ferramentas que buscavam restaurar o antigo Menu Iniciar ou adaptar o sistema para um uso mais tradicional com mouse e teclado.

Windows 10 (2015) – Sistema “como serviço”, com atualizações constantes. Unificou plataformas.

nível Básico reúne um conjunto limitado de informações que são fundamentais para compreender o dispositivo e sua configuração, incluindo: informações básicas do dispositivo, informações relacionadas à qualidade, compatibilidade de aplicativo e Microsoft Store. Quando o nível estiver definido como Básico, ele também incluirá as informações do nível de Segurança.

Windows 11 (2021) – Visual mais moderno, com foco em produtividade, jogos e integração com apps Android.

O Windows 11, lançado em 2021, apresenta um visual mais moderno e minimalista, com foco em produtividade, jogos e integração com apps Android.

Debian 1993

O Projeto Debian é uma associação de indivíduos que têm como causa comum criar um sistema operacional livre. O sistema operacional que criamos é chamado Debian. Um sistema operacional é o conjunto de programas básicos e utilitários que fazem seu computador funcionar. No núcleo do sistema operacional está o kernel. O kernel é o programa mais fundamental no computador e faz todas as operações mais básicas, permitindo que você execute outros programas.

Ubuntu 2004

O Ubuntu 20.04 LTS (Focal Fossa) foi lançado em 23 de abril de 2020, aprimorando a segurança corporativa sistêmica com medidas de autoproteção do kernel, WireGuard VPN, Secure Boot para proteção contra rootkits e FIDO para autenticação sem senha, entre outros.

Fedora 2003

Fedora (conhecido como Fedora Core antes da versão 7) é um sistema operacional(pt-BR) ou sistema operativo(pt-PT?) Linux. O sistema operacional Fedora Linux é software livre e de código aberto, e os programas disponíveis dentro de seu repositório de programas também são programas livres que aderem a uma licença livre.

Red Hat Enterprise Linux (RHEL) 2000

Red Hat Enterprise Linux (RHEL) 2000 refere-se à versão do Red Hat Linux lançada no ano 2000. Era uma distribuição Linux popular, especialmente para servidores, antes da Red Hat migrar para um modelo baseado em assinatura e introduzir o RHEL como seu principal produto. Antes do RHEL, o Red Hat Linux era um produto em caixa vendido em lojas de varejo, semelhante ao Microsoft Windows ou ao Lotus Notes.

CentOS (descontinuado como conhecíamos, mas ainda relevante historicamente)

O CentOS Linux é uma distribuição Linux gratuita e com suporte da comunidade, desenvolvida, distribuída e mantida pelos colaboradores do CentOS Project. Ele é derivado do Red Hat® Enterprise Linux® e foi lançado no mesmo ciclo do Red Hat Enterprise Linux. O CentOS Linux não é uma solução da Red Hat e não tem o suporte da Red Hat.

Linux Mint 2006

Linux Mint é uma distribuição Linux irlandesa. Possui duas versões: uma baseada em Ubuntu (com o qual é totalmente compatível e partilha os mesmos repositórios) e outra versão baseada em Debian. Suporta muitos idiomas, incluindo a língua portuguesa, e utiliza o Cinnamon como seu principal ambiente de desktop.

Arch Linux

Arch Linux, ou Arch (pronúncia em inglês: [ɑːrtʃ]),[3] é uma distribuição Linux para computadores com arquitetura x86-64. Desenvolvido inicialmente pelo canadense Judd Vinet, esse sistema operacional se apresenta de maneira diferente de outros, como Windows e MacOS. Além de ser composto predominantemente por software livre e de código aberto, ele envolve contribuições da comunidade.

Mac OS X 10.0 – Cheetah (2001)

Mac OS X 10.0 "Cheetah", foi a primeira versão do sistema operacional Mac OS X, para desktop e servidor da Apple Inc.. Mac OS X v10.0 foi lançado no dia 24 de março de 2001 por um preço de US$ 129.95. Substituiu a versão beta Público (internamente denominada "Kodiak") do Mac OS e veio antes do Mac OS X v10.1.

Mac OS X 10.4 – Tiger (2005)

O Mac OS X 10.4 "Tiger", foi o quinto lançamento do Mac OS X. O Tiger foi liberado ao público em 29 de abril de 2005 como o sucessor do Mac OS X 10.3 "Panther", que foi lançado 18 meses antes, sendo substituido pelo Mac OS X 10.5 "Leopard", em 26 de outubro de 2007, após 30 meses, tornando-se o Mac OS X de mais longa versão. Alguns dos novos recursos incluem um rápido sistema de busca chamado Spotlight, uma nova versão do navegador Safari, Dashboard, um novo tema unificado, e suporte aperfeiçoado para uso em processadores 64 bits.

Mac OS X 10.5 – Leopard (2007)

é o sexto grande lançamento do macOS , o sistema operacional de desktop e servidor da Apple para computadores Macintosh . O Leopard foi lançado em 26 de outubro de 2007 como sucessor do Mac OS X Tiger e está disponível em duas edições: uma versão desktop adequada para computadores pessoais e uma versão de servidor , o Mac OS X Server . Ele era vendido por US$ 129 [ 2 ] para a versão desktop e US$ 499 para o Server. [ 8 ] O Leopard foi substituído pelo Mac OS X Snow Leopard (versão 10.6) em 2009.

Mac OS X 10.7 – Lion (2011)

O Lion foi lançado para fabricação em 1º de julho de 2011, [ 11 ] seguido por seu lançamento final pela Mac App Store em 20 de julho de 2011. A Apple relatou mais de um milhão de vendas do Lion no primeiro dia de seu lançamento. [ 12 ] Em outubro de 2011 , o OS X Lion vendeu mais de seis milhões de cópias em todo o mundo.

macOS 10.10 – Yosemite (2014)

macOS 10.10 Yosemite foi a décima primeira versão principal do macOS (anteriormente OS X), lançado pela Apple em 2014. Foi anunciado na WWDC de 2014 e lançado para todos os usuários de Mac em 16 de outubro de 2014, por download na App Store. O Yosemite trouxe uma mudança visual, com uma interface mais moderna e plana, e também melhorias de desempenho e segurança.

macOS 10.15 – Catalina (2019)

O macOS Catalina (versão 10.15) é a décima sexta versão principal do macOS, o sistema operacional para computadores Mac da Apple. Foi lançado em outubro de 2019 e trouxe diversas novidades, incluindo a remoção do suporte a aplicativos de 32 bits, a introdução do Apple Arcade, e a unificação dos aplicativos Apple Music, Apple Podcasts e Apple TV.

macOS 11 – Big Sur (2020)

O macOS Big Sur (versão 11.0) foi lançado pela Apple em 12 de novembro de 2020, apresentando uma nova interface de usuário com elementos translúcidos e ícones mais uniformes, além de melhorias em aplicativos como Safari, Mensagens e Mapas.

macOS 12 – Monterey (2021)

O macOS Monterey (versão 12) é a décima oitava versão principal do sistema operacional macOS da Apple, lançado em 25 de outubro de 2021. Foi anunciado na WWDC 2021 em 7 de junho de 2021 e sucessor do macOS Big Sur (versão 11).

macOS 13 – Ventura (2022)

O macOS Ventura (versão 13) foi lançado pela Apple em 2022, sendo o sucessor do macOS Monterey. Ele traz várias melhorias e novos recursos, incluindo o Stage Manager para organização de janelas, a Câmera de Continuidade para usar o iPhone como webcam e aprimoramentos em aplicativos como Safari e Mail.

macOS 14 – Sonoma (2023)

O macOS Sonoma (versão 14) é a mais recente versão do sistema operacional da Apple para Mac, lançada em 26 de setembro de 2023. Foi anunciado na WWDC 2023 em junho de 2023. Este sistema operacional é conhecido por suas novas funcionalidades, como widgets personalizados e melhorias no modo de jogo, entre outras.
